# Cybersecurity Tasks Project (Task 1, Task 2, Task 3)

This project contains **three Python security tasks**:

1. Task 1 - Authentication System** (`task1_auth.py`)
2. Task 2 - SIEM Log Analyzer** (`task2_siem.py`)
3. Task 3 - Malware Analysis & Forensics Tool** (`task3_forensics.py`)

This README explains how to set up and run all three tasks properly.

---

## 1) Project Requirements

- Linux environment (recommended)
- Python 3.8+
- Optional but recommended: Python virtual environment

# Check Python version
```bash
python3 --version
```

---

## 2) Setup Instructions

From project directory (`/home/manish`), create and activate a virtual environment:

# bash
python3 -m venv .venv
source .venv/bin/activate
```

# Upgrade pip:

# bash
pip install --upgrade pip

Install dependency used by Task 3 (EXIF metadata extraction):

bash
pip install Pillow

If Pillow is not installed, Task 3 still runs, but EXIF/GPS extraction is skipped and warning is shown.

## 3) Task 1 - Secure Authentication System

**File:** "task1_auth.py"

### What it does
- User registration and login (CLI menu)
- Password strength validation:
  - Minimum 12 characters
  - Uppercase + lowercase + number + special character
- Password hashing using PBKDF2-HMAC-SHA256
- Stores credentials in `users.txt`
- Login delay (2sec)on failure to slow brute-force attempts

### Run Task 1
```bash
"python3 task1_auth.py"


### Menu options
- `1` Register
- `2` Login
- `3` Exit

### Output/Storage
- Creates/uses `users.txt` in current directory.

---

## 4) Task 2 - SIEM Log Analysis Tool

**File:** `task2_siem.py`

### What it does
- Reads authentication logs (like SSH auth logs)
- Detects failed login attempts by IP
- Counts failed attempts per IP
- Flags suspicious IPs above threshold
- Prints alert report
- Saves report to `malicious_ips.json`

### Run Task 2
```bash
python3 task2_siem.py <logfile> <threshold>
```

# Example
```bash
python3 task2_siem.py auth.log 5
```

# Arguments
- `<logfile>`: path to log file
- `<threshold>`: integer failed-attempt cutoff like  (,5)

# Output
- Console alert/details
- JSON file: `malicious_ips.json`

# 5) Task 3 - Malware Analysis and Forensics Tool

File:= `task3_forensics.py`

# What it does
- Interactive malware scanning menu
- Scans single file or full folder
- Detects true file type using magic bytes
- Calculates SHA-256 hash and compares with known malicious signatures
- Optional EXIF/GPS metadata extraction (images only, via Pillow)
- Quarantines detected malicious files into vault
- Logs incidents into scan log file
- Includes quick MD5 test scan option for `fake_virus.txt`

# Run Task 3
bash
python3 task3_forensics.py


# Task 3 Menu
- 1 Scan Single File
- 2 Scan Entire Folder
- 3 Show Quarantine Vault
- 4 Show Scan Log
- 5 Exit
- 6 Quick MD5 Test Scan (`fake_virus.txt`)

# Important paths used in script
- Quarantine vault: `/home/manish/QUARANTINE_VAULT`
- Scan log: `/home/manish/scan_log.txt`

# Output
- Console scan results
- Quarantined suspicious files moved to vault
- Event log appended in `scan_log.txt`

# 8) Troubleshooting

### `ModuleNotFoundError: No module named 'PIL'`
Install Pillow:
```bash
pip install Pillow
```

### Task 2 shows usage error
Use exactly:
```bash
python3 task2_siem.py <logfile> <threshold>
```

### No suspicious IPs found in Task 2
- Confirm log file has lines matching failed password format
- Lower threshold value for testing

### Task 3 cannot find file/folder
- Provide full path, e.g. `/home/manish/somefile.bin`
- Check permissions for scanned files
